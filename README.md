## snake

A simple [snake clone](https://en.wikipedia.org/wiki/Snake_(video_game_genre)) made for fun and learning.

## usage

```console
$ git clone https://github.com/vilhelmbergsoe/snake && cd snake/
$ make && ./snake
or
$ make run
[score]: 9

```

try and beat my record (44) and have fun!

## keybindings

`w` => up

`s` => down

`a` => left

`d` => right

`p` => pause / unpause

`q` => quit game
